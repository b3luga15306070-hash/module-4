Простое консольное приложение на Python для ведения списка прочитанных книг.

---

##  Возможности

- Добавление книги (автор, название, оценка, дата)
- Проверка на дубликаты (автор + название)
- Просмотр всех книг
- Подсчёт средней оценки
- Удаление книги
- Сохранение данных в JSON файл

---

##  Технологии

- Python 3
- JSON (для хранения данных)
- Работа с файлами (os, json)
